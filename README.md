# uddi-enhanced

A collection of [Tampermonkey](https://www.tampermonkey.net) Userscripts to improve Infoblox UDDI experience

# Installation

1. Get the Tampermonkey Extension for your [Chrome](https://chromewebstore.google.com/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo) or [Firefox](https://addons.mozilla.org/de/firefox/addon/tampermonkey/) browser
2. Click the link of the desired scripts below
3. Extension should open the install dialogue automatically
4. Hit "install"

## Userscripts

### Dynamic Tab Title
Replaces the default "Infoblox Portal" text in the browser tab with something related to the current open page in that tab.
Useful if you're working with multiple tabs.

### Disable Dangerous CSV Import Options
Disables the most dangerous radio buttons in the CSV Import option selection.  
Clicking the "I know what I'm doing"-button to the right will enable the radio button to be checked again.
